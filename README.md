# Thor Ragnarok

This is a Game Boy Advance game I made for CS 2110 HW9.

### Installation
	Run:
	make vba

### Controls
	START = Enter, to start the game
	SELECT = Backspace, to reset to start screen
	UP = Up
	DOWN = Down
	LEFT = Left
	RIGHT = Right
	A = Z

### How to Win
  Defeat Loki, the Hulk, and Hela.

### How to Lose
  Be defeated by Loki, the Hulk, or Hela.
